

# Personal Portfolio Website for P. Venkatesh

## Overview
A clean, modern, and responsive portfolio website for a first-year frontend developer student. The site will have a professional blue/gray color palette, smooth animations, and a mobile-first design.

---

## Pages & Sections (Single-Page Layout)

### 1. Hero Section
- Large greeting with name "P. Venkatesh" and role "Frontend Developer"
- Tagline: "First-year student learning and building web projects"
- Call-to-action buttons: "View Projects" and "Contact Me"
- Subtle fade-in animation on load

### 2. About Me Section
- Brief paragraph about being a first-year student and aspiring frontend developer
- Mention of interest in web development and learning new technologies
- Current skills overview (HTML, CSS, C, Python)
- Simple, friendly, and professional tone

### 3. Skills Section
- Visual skill cards with icons for each technology
- Two categories: "Known" (HTML, CSS, C Language) and "Currently Learning" (Python)
- Progress indicators or badges to show proficiency level

### 4. Projects Section
- Three project cards in a responsive grid layout
- Each card includes: title, description, tech tags, and GitHub/Live Demo buttons (placeholder links)
- Projects: Personal Portfolio, Basic Calculator, Learning Project

### 5. Contact Section
- Contact info: email, GitHub, LinkedIn (all placeholder links)
- Simple contact form with name, email, and message fields (frontend-only for now)
- Note that details can be updated later

### 6. Sticky Navigation Bar
- Fixed top navigation with smooth scroll links to each section
- Mobile hamburger menu
- Clean, minimal design with the blue/gray theme

---

## Design & UX
- **Color palette**: Professional blue and gray tones
- **Typography**: Clean, readable fonts with proper spacing
- **Animations**: Subtle fade-in on scroll for each section
- **Responsive**: Mobile-first layout that looks great on all screen sizes
- **Light theme** with modern minimal styling
